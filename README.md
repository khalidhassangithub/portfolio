# Midnight Premium Portfolio

A state-of-the-art, professional portfolio designed for Data Science Engineers and Android Developers. Built with **Next.js 15**, **Tailwind CSS**, and **Framer Motion**, this project features a sophisticated "Midnight Premium" aesthetic with cinematic animations and glassmorphism.
